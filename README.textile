h1. A Few Select Chef Cookbooks

Here you'll find a few cookbooks intended to aid in my "attempt to build a graphite server":http://github.com/gorsuch/sandbox_stem.

h2. Caveats

* many things are installed via the "bash" resource, which means that they are run every time chef is run.  Not terrible, but not efficient.  These things should be moved to debian packages
* we skip adding an administrative user to the web interface (based on Django)
